# Random GIF Generator

This project is a Random GIF Generator built using ReactJS and Hooks. It allows users to discover and display random animated GIFs, providing an interactive and entertaining experience.

## Features

* Random GIF fetching from the GIPHY API.
* Clean and intuitive user interface.
* Search functionality to find specific GIFs based on keywords or tags.
* Responsive design for seamless usage across different devices.


## Technologies Used

* HTML
* CSS
* React JS
* GIPHY API

## How to Use

Follow The 
Link :- https://i-pranjalmishra.github.io/RandomGIF-ReactJS/

#### OR

1. Clone the repository to your local machine or download the source code.
2. Open the terminal and install npm
3. Now write "npm start" on terminal.
4. Webpage will be live.
